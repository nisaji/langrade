# Installation

## Requirements

- Python 3.9+
- API key for the chosen LLM provider (OpenAI, Anthropic, or Google)

## Using pip

Install Langrade using pip:

```bash
pip install langrade
```

## From source

To install Langrade from source:

```bash
git clone https://github.com/nisaji/langrade.git
cd langrade
pip install .
```
